# Rocky Three-Level Retrieval Plan
Platform: cross_platform
Level: system
Topic: retrieval

Meta retrieval: campaign/objective + ad-set optimization + audience/placements + ad/creative + measurement + execution rules.
Google retrieval: campaign type/bidding + ad-group or asset-group knowledge + ad/assets + measurement + execution rules.

Use metadata filters for platform + level + topic before semantic ranking. Do not use one generic top-k query for the entire campaign plan.
