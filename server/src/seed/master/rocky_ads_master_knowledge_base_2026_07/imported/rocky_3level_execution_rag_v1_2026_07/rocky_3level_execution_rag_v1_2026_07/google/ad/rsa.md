# Google Responsive Search Ads

Source: https://developers.google.com/google-ads/api/docs/responsive-search-ads/get-responsive-search-ads

Responsive Search Ads contain multiple headline and description assets that Google can combine. The API exposes RSA headlines and descriptions on the ad resource.

Create materially distinct, truthful assets tied to search intent. Never fabricate claims, prices or credentials. Validate exact current asset limits before submission.
