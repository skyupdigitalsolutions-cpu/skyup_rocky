# Meta Pixel and Conversions API
Platform: meta_ads
Level: ad
Topic: measurement
Source: https://www.facebook.com/business/help/AboutConversionsAPI
Verified: 2026-07-24

Conversions API connects business marketing data to Meta systems for optimization and measurement and supports website, app, offline and messaging event use cases. Meta recommends considering CAPI alongside Pixel for website events.

Rocky must verify event source, desired event, browser/server deduplication where relevant, and downstream signal quality. CAPI is not a privacy-policy bypass. Never claim Pixel/CAPI is healthy without connected evidence.
