# Source Integrity Rules
Knowledge version: 2026-07

Labels:
IIMB_PUBLIC_SOURCE = facts directly supported by publicly accessible IIM Bangalore/IIMBx pages.
IIMB_DERIVED_SYNTHESIS = original Rocky strategy inference based on those public themes.
META_OFFICIAL_RULE = retrieve from separate current Meta official knowledge base.
ACCOUNT_DATA = connected Skyup advertising/account data.
CLIENT_DATA = must remain isolated according to Rocky's data architecture.

Never label authored Rocky advice as an IIM Bangalore recommendation.
Never reproduce paywalled/private course notes, copyrighted textbooks or classroom materials into the vector database without appropriate rights.
Never infer that IIMB endorses Rocky, Skyup or a campaign.
