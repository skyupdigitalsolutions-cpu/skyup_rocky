# Google Ads API — Targeting Levels
Platform: google_ads
Level: campaign_ad_group
Topic: targeting_api
Source: https://developers.google.com/google-ads/api/docs/targeting/overview
Verified: 2026-07-24

Google Ads API targeting can operate at customer, campaign and ad-group levels depending on criterion type. Criteria may support targeting, exclusion, bid changes, or only a subset of those operations.

Rocky must not assume every criterion is valid at every level. The connected API version is the final authority.
