# Google Enhanced Conversions — 2026 Migration
Platform: google_ads
Level: measurement
Topic: enhanced_conversions_2026
Sources:
- https://support.google.com/google-ads/answer/14664077?hl=en
- https://support.google.com/google-ads/answer/16884284?hl=en
Verified: 2026-07-24

Google is unifying enhanced conversions for web and leads. From April 2026 Google can accept user-provided data simultaneously from website tags, Data Manager and API connections. Google states that from June 15, 2026, offline-conversion import and enhanced-conversions-for-leads uploads are migrated to Data Manager API and blocked in Google Ads API except documented legacy circumstances.

Rocky's integration code must not be designed around an obsolete upload path. Measurement implementation should follow the current Data Manager/API documentation.
