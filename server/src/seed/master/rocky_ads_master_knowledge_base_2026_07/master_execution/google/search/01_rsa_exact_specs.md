# Google Search — Responsive Search Ad Exact Specs
Platform: google_ads
Level: ad
Topic: rsa_specs
Source: https://support.google.com/google-ads/answer/17092074?hl=en
Verified: 2026-07-24

Current official Search campaign specifications list Responsive Search Ads with headlines up to 30 characters, 1–15 headline assets, descriptions up to 90 characters, 1–4 description assets, and a final URL up to 2,048 characters.

Rocky must validate generated RSA text against these limits before submission. Character validation belongs in deterministic code as well as RAG guidance.
