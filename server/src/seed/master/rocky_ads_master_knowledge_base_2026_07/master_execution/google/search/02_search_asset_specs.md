# Google Search — Search Asset Specifications
Platform: google_ads
Level: ad
Topic: search_asset_specs
Source: https://support.google.com/google-ads/answer/17092074?hl=en
Verified: 2026-07-24

Google's current Search specifications document business information and ad assets in addition to RSA text. Current documented examples include a business name up to 25 characters and a square logo recommendation of 1200×1200 pixels with a documented minimum of 128×128. Image assets include square 1:1 and horizontal 1.91:1 formats with current documented size guidance.

Rocky should use exact current specs only for the asset type being created and should validate uploaded files before API creation.
