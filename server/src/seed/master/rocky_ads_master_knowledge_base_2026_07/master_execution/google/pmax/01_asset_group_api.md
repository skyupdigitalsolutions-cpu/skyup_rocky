# Google Performance Max — Asset Group API
Platform: google_ads
Level: asset_group
Topic: asset_group_api
Source: https://developers.google.com/google-ads/api/reference/rpc/v24/AssetGroup
Verified: 2026-07-24

An AssetGroup belongs to a campaign and links assets through AssetGroupAsset and signals through AssetGroupSignal. Current API fields include name, final URLs, mobile URLs, path fields, status, ad strength, asset coverage and primary-status diagnostics.

Rocky should use asset-group diagnostics as live account evidence after creation. Static RAG cannot predict ad strength or asset coverage.
