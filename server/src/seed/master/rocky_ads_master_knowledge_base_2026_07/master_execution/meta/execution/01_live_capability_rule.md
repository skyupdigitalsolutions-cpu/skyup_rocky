# Meta — Live Capability Rule
Platform: meta_ads
Level: system
Topic: live_validation

Meta RAG provides objective, audience, placement, measurement and creative reasoning, but exact IDs, available optimization goals, destination combinations, account eligibility, placements and API fields must be resolved from the connected Marketing API/account.

Rocky must never convert a human-readable targeting idea into an executable Meta ID unless it came from a live supported lookup or verified stored mapping.
